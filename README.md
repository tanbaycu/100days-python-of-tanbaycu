```bash
Python 3.8+ (Khuyến nghị Python 3.11+)
Git

```

### 💻 Cài Đặt
```bash

git clone https://github.com/tanbaycu/100days-Python-of-tanbaycu.git

# Chuyển vào thư mục
cd 100days-Python-of-tanbaycu

# Cài đặt dependencies
pip install -r requirements.txt

cd giai-doan-1-co-ban/ngay-01-02-hello-world
python hello.py
```
